# Series_Sinteticas
Ferramentas em Python para geração e avaliação de séries sintéticas aplicadas ao planejamento energético.

## Sobre o repositório

Este repositório reúne modelos e rotinas em **Python** para geração de séries sintéticas de variáveis relevantes ao **planejamento energético**, como afluências, carga, entre outras.

O objetivo é oferecer um conjunto de ferramentas modulares para:

- Modelagem estatística e estocástica de séries temporais;
- Geração de cenários sintéticos para estudos de médio e longo prazo;
- Apoio a modelos de planejamento e operação de sistemas de energia.
